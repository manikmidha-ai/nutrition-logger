Firebase Setup (free, 3 min)

1
Create a free Firebase project
Go to console.firebase.google.com → Add project → any name → Create project
2
Create a Realtime Database
Build → Realtime Database → Create Database → any region → Start in test mode → Enable
3
Set rules to allow access
Rules tab → set .read and .write to true → Publish
4
Copy your Database URL
Data tab — the URL shown (e.g. https://your-project.asia-southeast1.firebasedatabase.app)
5
Add your Anthropic API key
Go to console.anthropic.com → API Keys → Create key → paste below. This stays on your device only.
