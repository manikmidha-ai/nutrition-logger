Firebase Setup (free, 3 min)

1
Create a free Firebase project
Go to console.firebase.google.com → Add project → any name → Continue → Create project
2
Create a Realtime Database
In your project → Build → Realtime Database → Create Database → choose any region → Start in test mode → Enable
3
Set rules to allow access
Click the Rules tab → set both .read and .write to true → Publish
4
Copy your Database URL
Click the Data tab — the URL at the top (e.g. https://your-project-default-rtdb.asia-southeast1.firebasedatabase.app)

Your data is stored privately in your own Firebase account. Free tier is more than enough for personal use.
